# homebrew

> mac 上的包管理工具

## 一、homebrew 自身相关

**升级 homebrew**
`brew update`

## 二、包相关

**安装包**
`brew install [app_name]`

**更新包** 不指定 app 时默认更新所有包
`brew upgrade [app_name]` 

**查看包信息**
`brew info [app_name]`

**查看本地已安装的包**

`brew lsit`

## 三、启动 homebrew 安装的服务

启动：`brew service start xxx` 

停止：`brew service stop xxx`
