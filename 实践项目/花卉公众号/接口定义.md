## 07购物车

- 获取购物车列表     
  参数： 用户id

```
返回值： 商品购物车列表
[{
    id: 1111 
    product: '绿箩', //商品名称
    price: 100,  // 商品单价
    images: [],  // 商品图片
    sum: 1000  // 数量
 }, {
    id: 1111 
    product: '绿箩', //商品名称
    price: 100,  // 商品单价
    images: [],  // 商品图片
    sum: 1000  // 数量
}]
```

- 删除购物车记录      
  参数： 记录id（数组，删除多条记录）    
  返回值：无
