# QWIK

> qwik 是一款 angular 支付推出的 html 优先的前端框架。特点是快，将 js 的加载放到交互时，避免一开始的 js 下载和解析动作。首屏达到极速渲染，并且极速可交互。

官方仓库：https://github.com/BuilderIO/qwik

官方文档：https://qwik.builder.io/docs/overview
