# CSS 工程化方案

在项目中 CSS 由于没有作用域可言，容易导致很多问题。现代的前端开发中解决这个问题大致可以分为以下三类方案：

1. **CSS 命名方法论**：通过人工的方式来约定命名规则。
2. **CSS Modules**：一个 CSS 文件就是一个独立的模块。
3. **CSS-in-JS**：在 JS 中写 CSS。
